# rufyn-technologies

Politica de privacidade estatica do Chapterly pronta para publicar no GitHub Pages.

## Publicacao no GitHub Pages

1. Envie este repositorio para o GitHub.
2. No repositorio, abra `Settings` > `Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch principal e a pasta `/ (root)`.
5. Salve e aguarde a URL publica ser gerada.

O arquivo principal publicado sera `index.html`.
